# Dashboards

This folder contains the Databricks SQL dashboard assets for the SEC Smart Money capstone.

## Files

- `dashboard_config.yml`: dashboard layout + tile/chart/filter configuration
- `dashboard_queries.sql`: SQL queries powering KPIs, charts, tables, and filters

## Documentation

- Full user guide: `docs/DASHBOARD_DOCUMENTATION.md`

## Notes

- Queries reference the example catalog `fintech_analytics`. Update catalog/schema names if your workspace differs.
